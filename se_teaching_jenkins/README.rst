Jenkins 
=======

1. Uruchom jenkins-a:

   ::

     make build_jenkins
     make run_jenkins

2. Otwórz w przeglądarce 127.0.0.1:8080, jeśli zostaniesz poproszony o hasło dla admina, wybierz:

   ::

     cat jenkins/secrets/initialAdminPassword

3. Wybierz *Suggested plugins*.
